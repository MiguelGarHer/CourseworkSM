#Report: Top N populated countries in the world

---
## CHARACTERISTIC INFORMATION

---
### Goal in Context

As the company, I want a report that shows the top N populated countries in the world.

### Scope

Company.

### Level

Primary task.

### Preconditions

Database contains cities, region and population data.

### Success End Condition

Top N populated countries in the world.

### Failed End Condition

No list is produced.

### Primary Actor

Organisation.

### Trigger

A request from an organisation to generate a report.

## MAIN SUCCESS SCENARIO

1. Access to the population information.
2. Write a SQL code to create a list with the request.
3. Provide the report.
